  • Rails Console
For those who are using ubuntu or Mac, under your home directory put this script in ~/.irbrc

script_console_running = ENV.include?('RAILS_ENV') && IRB.conf[:LOAD_MODULES] && IRB.conf[:LOAD_MODULES].include?('console_with_helpers')
rails_running = ENV.include?('RAILS_ENV') && !(IRB.conf[:LOAD_MODULES] && IRB.conf[:LOAD_MODULES].include?('console_with_helpers'))
irb_standalone_running = !script_console_running && !rails_running

require "rubygems"

if script_console_running
  require 'hirb'
  Hirb::View.enable

  require 'logger'
  Object.const_set(:RAILS_DEFAULT_LOGGER, Logger.new(STDOUT))
end

  • RVM:
  • Install rvm as said from any of the following resources:
    • http://beginrescueend.com/rvm/install/
    • http://railscasts.com/episodes/200-rails-3-beta-and-rvm
  • Install new rvm gemset:
  • Install rvm ruby version `rvm install 1.9.2`
  • Use new ruby version `rvm use 1.9.2`
  • List all ruby versions `rvm list`
  • Create new gemset inside a specific ruby version:
    • move to the rvm ruby version `rvm use 1.9.2`
    • create the gemset `rvm gemset create my_project_name`
    • switch to the gemset `rvm 1.9.2@my_project_name`
  • To avoid manually switch, you can create .rvmrc file in the project root with the following contents, on the terminal
    • echo "rvm 1.9.2@my_project_name" > .rvmrc
      If the rvm doesn't feel the existence of .rvmrc file, please read the following (extracted from here):
      As of RVM 1.7.0 project .rvmrc files are opt-in by defualt (therefore off). To order to enable this feature set the following value in either /etc/rvmrc or ~/.rvmrc:
      rvm_project_rvmrc=1
